# Security Policy

This repository is a portfolio project and does not process live vendor, customer, or regulated data.

## Reporting

If you notice a security issue in the application shell or development workflow, please open a private report through GitHub Security Advisories or contact the repository owner through official channels listed on [Kinetic Gain](https://kineticgain.com/).

## Scope

- No production credentials should be committed.
- Demo data should remain synthetic.
- Third-party package updates should continue through Dependabot and CI validation.
